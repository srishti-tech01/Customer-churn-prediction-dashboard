# Customer Churn Prediction Dashboard

An interactive Machine Learning dashboard that predicts customer churn using Logistic Regression and Streamlit.

## Features
- CSV File Upload
- Data Cleaning & Preprocessing
- Logistic Regression Model
- Accuracy Score
- Confusion Matrix Visualization
- Interactive UI Dashboard

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-Learn
- Streamlit
- Matplotlib & Seaborn

## How to Run
pip install -r requirements.txt
streamlit run app.py

## Project Goal
To analyze customer data and predict whether a customer is likely to churn, providing businesses insights to improve retention.
